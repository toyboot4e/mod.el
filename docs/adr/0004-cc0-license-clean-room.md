# CC0-1.0 license, clean-room implementation

mod.el is CC0-1.0, matching aim-mode.el, instead of the elisp-ecosystem default GPL. This forbids copying code from the GPL libraries this project was designed against — and we studied leaf.el's source in detail during the design session (its keyword table, recursive handler engine, and deferral mechanism), so the line must be explicit: architecture, ideas, and observed behavior are free to use; reproducing their code is a license violation. The same applies to use-package, setup.el, and Emacs internals. The trade-off accepted: defmod's engine is written from scratch even where leaf has a working GPL implementation to lift from — affordable because the fixed-skeleton design (ADR-0003) shares no code shape with leaf's engine anyway.
